CodeClock is a lightweight time-tracking extension for Visual Studio Code that automatically tracks how much time you spend working in each workspace.

It runs in the background, pauses when you're inactive, and gives you a clear overview of your coding time directly inside VS Code.

## üåü Features

- **Automatic time tracking**: Starts tracking when a workspace is opened and works per project.
- **Idle detection**: Pauses the timer after a period of inactivity and resumes when you return.
- **Session history**: Tracks individual coding sessions with start time, end time, and duration.
- **Sidebar view**: Displays live timer, total project time, and session history.
- **Status bar integration**: Shows current project name and elapsed time at a glance.
- **Local storage only**: All data is stored locally using VS Code storage.
